# Privacy Policy for YouTube Hover Actions

Last updated: April 29, 2026

YouTube Hover Actions does not collect, store, sell, share, or transmit user data.

The extension runs locally in the browser on YouTube pages. It adds small hover buttons to YouTube video cards and uses YouTube's own native menu actions when the user clicks those buttons.

## Data Collection

This extension does not collect:

- Personally identifiable information
- Browsing history
- Search history
- YouTube account information
- Video watch history
- Page content
- Analytics or usage events

## Data Storage

This extension does not store user data locally or remotely.

## Data Sharing

This extension does not share user data with the developer, third parties, analytics providers, advertisers, or external services.

## Remote Code

This extension does not use remote code. All JavaScript and CSS files are packaged inside the extension and run locally in the browser.

## Contact

For questions about this privacy policy, contact the publisher using the contact email listed on the Chrome Web Store page.
