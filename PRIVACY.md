# Privacy Policy for YouTube Hover Actions

Last updated: June 27, 2026

YouTube Hover Actions does not collect, store, sell, share, or transmit user data.

The extension runs locally in the browser on YouTube pages. It adds small hover buttons to YouTube video cards, uses YouTube's own native menu actions, and offers a dynamic configuration panel.

## Data Collection

This extension does not collect:

- Personally identifiable information
- Browsing history
- Search history
- YouTube account information
- Video watch history
- Page content
- Analytics or usage events

## Data Storage

This extension stores user configuration settings (such as toggling "Hide YouTube Shorts" and "Show Hover Actions") locally in the browser using the `chrome.storage.local` API. No personal or tracking data is stored.

## Data Sharing

This extension does not share user data with the developer, third parties, analytics providers, advertisers, or external services.

## Remote Code

This extension does not use remote code. All JavaScript, CSS, and styling assets are packaged inside the extension and run locally in the browser.

## Contact

For questions about this privacy policy, contact the publisher using the contact email listed on the Chrome Web Store page.
